# Medical Center Management System (MCMS)

MCMS is a web-based Medical Center Management System built with PHP and MySQL. The system helps a medical center manage staff login, patients, appointments, doctor consultations, billing, reports, and user roles.

## Project Features

- User login system with role-based access
- Admin dashboard
- Patient management
- Appointment management
- Appointment check-in and cancellation
- Doctor consultation page
- Billing and payment records
- Admin user management
- Reports page
- Audit log for tracking system actions

## Tools and Technologies Used

- PHP
- MySQL
- Apache Server
- XAMPP
- PDO for database connection
- HTML
- CSS
- JavaScript
- phpMyAdmin

## Project Setup Instructions

### 1. Install XAMPP

Download and install XAMPP on your computer. XAMPP is used to run Apache and MySQL locally.

### 2. Move the Project Folder

Copy the project folder into the XAMPP `htdocs` folder.

Example path on Windows:

```text
C:\xampp\htdocs\mcms
